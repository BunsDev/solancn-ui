# Solancn UI (Alternative) without shadcn/ui

## CLI Usage

Initialize your project with the CLI:

```bash
npx solancn@latest init
```
```md
npx solancn@latest init
✔ SolancnUI initialized successfully!

🌐 Registry URL: https://ui.solancn.com/registry

📦 Dependencies installed:
  - clsx
  - tailwind-merge

📁 Directories created:
  - lib/
  - components/

✨ Ready to add components!
```

To see all available components:

```bash
npx solancn@latest show
```

```md
npx solancn@latest show
Need to install the following packages:
solancn@latest
Ok to proceed? (y)

✔ SolancnUI Available Components:

🧩 Components available for installation:
Last updated: 6/27/2025, 1:08:46 PM
Total components: 5

  1. accordion
  2. button
  3. tabs-classic
  4. tabs-fancy
  5. tabs

📖 For installation, type:
  npx solancn@latest add <component-name>

💡 Example:
  npx solancn@latest add button

```

## Initialize with a custom registry URL

```bash
npx solancn@latest init --url https://ui.solancn.com/registry
```

## Add a component (uses saved URL)

```bash
npx solancn@latest add button
```
	
```md
npx solancn@latest add button
✔ Button added successfully!
```


## Add a component with a custom URL (one-time)

```bash
npx solancn@latest add button --url https://different-registry.com/registry
```

## Configure the default registry URL

```bash
npx solancn@latest config --url https://ui.solancn.com/registry
```
